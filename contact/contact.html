Add predictor on your own website

Adding the NMR predicting tool on your website server is not only FREE but also very easy. Just add the following line in which the last parameter is the URL encoded SMILES code:

http://www.nmrdb.org/predictor?smiles=

For example:

NMR spectrum of cholesterol

You can also directly submit a molfile, check the source code of this page to see how easy it is!
